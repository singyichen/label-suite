# 訪談指引：Annotator（工讀生）

**角色說明**：執行標注任務的大學生或研究生，依照研究生指派的任務完成 NLP 資料標記工作。

---

## 訪談前準備

- 確認受訪者近期有執行過標注工作
- 準備筆記本或錄音設備（徵得同意後）
- 預計時間：45–60 分鐘

---

## 一、暖身與背景（5–10 分鐘）

| 編號 | 問題 |
|------|------|
| A01 | 請簡單介紹你自己，目前在哪個實驗室協助標注工作？ |
| A02 | 你做標注工作多久了？大概每週花多少時間在上面？ |
| A03 | 你做過哪些類型的標注任務？（文本分類、命名實體、情感分析等） |
| A04 | 你目前使用什麼工具或方式完成標注？（Label Studio、Excel、自製工具等） |

---

## 二、任務接收與理解（10 分鐘）

| 編號 | 問題 |
|------|------|
| A05 | 研究生通常用什麼方式告訴你要做什麼任務？（口頭、LINE、email、系統通知） |
| A06 | 你如何知道任務的截止日期和優先順序？ |
| A07 | 任務說明清楚嗎？有沒有過看不懂規則、需要反覆詢問的經驗？ |
| A08 | 你希望任務說明以什麼形式呈現最清楚？（文字、範例、影片） |

---

## 三、標注作業流程（15 分鐘）

| 編號 | 問題 |
|------|------|
| A09 | 描述一下你平常一次標注工作的完整流程，從接到任務到完成提交。 |
| A10 | 在標注過程中最常遇到什麼困難？（介面操作、規則模糊、資料品質） |
| A11 | 你有沒有遇過標注到一半系統出問題或資料消失的情況？ |
| A12 | 你每次能標多少筆資料？有沒有數量上的壓力？ |
| A13 | 完成標注後，你如何提交成果？有沒有需要額外確認或回報的步驟？ |

---

## 四、工時與薪資管理（10 分鐘）

| 編號 | 問題 |
|------|------|
| A14 | 你目前怎麼記錄工作時數？（系統自動、自己填 Excel、口頭回報） |
| A15 | 有沒有遇過工時計算不準確或有爭議的情況？ |
| A16 | 你怎麼知道自己這個月能領多少薪資？有清楚的薪資明細嗎？ |
| A17 | 你希望系統能自動記錄你的工時嗎？還是你偏好手動填寫？為什麼？ |

---

## 五、回饋與期望（10 分鐘）

| 編號 | 問題 |
|------|------|
| A18 | 你最希望標注系統改善哪一件事？ |
| A19 | 如果有一個介面能讓你看到自己的標注進度、工時和薪資，你覺得有幫助嗎？ |
| A20 | 你有沒有看過讓你覺得「這個設計很好用」的系統或工具？是什麼讓你這樣覺得？ |
| A21 | 還有什麼事情你想補充，或是你認為我沒問到但很重要的？ |

---

## 訪談後注意事項

- 確認受訪者是否同意錄音及資料使用方式
- 記錄檔命名：`YYYY-MM-DD-annotator-P0X.md`
- 記錄重點：工具使用習慣、痛點關鍵詞、對自動工時記錄的態度
